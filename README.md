## Manager de Contraseñas

La intención al generar esta app fue practicar e ir puliendo temas de React y Tipescript. Además de utilizar componentes de shadcn.

# A grandes rasgos la web te permite:

.- LLevar un control de las contraseñas de las distintas webs que puedas usar.
.- Guardar tanto contraseñas y números de tarjetas.
.- Generar contraseñas de hasta longitud 50 randomizadas, aplicando distintos elementos como mayusculas, caracteres especiales y demás.
